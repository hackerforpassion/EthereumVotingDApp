# Voting App built on Ethereum

### Dependencies:
- [Nodejs 5.0+](https://nodejs.org/en/)
- [Truffle](https://github.com/trufflesuite/truffle)
- [Ganache](http://truffleframework.com/ganache/)

## Setup
```
npm install -g truffle
npm install
npm run dev

```
## Run Test RPC
```
testrpc

```
## Deploy contract to TestNet
```
truffle migrate

```
